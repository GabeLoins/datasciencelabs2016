Functions and Data Types

In Python there are functions, such as print(), that are built-in as well as others that are user-defined. You will be creating lots of functions throughout the course. Every function block in Python begins with the keyword def and ends with parentheses. These parentheses can take parameters or arguments as necessary. The return statement exits a function. Functions can be executed by having another function call on them, or directly from the Python prompt. For example: 

def print_this(str):
  print str
  return

Numbers in Python are either integers or floats. An integer is a whole number, and floats are floating point numbers such as 3.65 and 52.3E-4.

A string is a sequence of characters. Strings in Python are immutable, which means that you cannot change them. They can be indexed, with the first character at index 0. A string can be lots of characters or a single character. Python includes functions that can operate on string objects. You can see these string functions here: https://docs.python.org/2/library/string.html#string-functions

Lists and Tuples

Python contains data types that are used to group values together. A list contains comma-separated values between square brackets. These values can be different types. Just like strings, you can index into an item of a list. Unlike strings, lists are mutable so you can change their content. For example, 

>>> my_list = [1, 2, 3, 4, 5]
my_list[3] = 7
>>> my_list
[1, 2, 3, 7, 5]
Lists are a flexible and extremely useful data type. Its functions include append(), sort(), reverse(), insert(), and more. Explore list functions here: https://docs.python.org/2/tutorial/introduction.html#lists

A very useful operation with lists is list comprehensions. This is an easy way to create lists where each element is the result of some operations. Read more about list comprehensions here: https://docs.python.org/2/tutorial/datastructures.html#list-comprehensions

You can iterate through lists using loops. An commonly-used option is the for loop, which can iterate over the items of lists and strings. For example: 

for letter in 'Hello':
  print letter
You can also write a for loop that iterates by sequence index instead of iterating through each item. For example: 
furniture = ['sofa', 'couch', 'dining table']
for i in range(len(furniture)):
  print furniture[i]

Tuples are another standard sequence data type in Python. They consist of values separated by commas. While they can seem similar to lists, tuples are used differently since they are immutable. This is what a tuple can look like: 

tup = ('chair', 'mirror', 'stove', 'shelf')
tup2 = (1, 2, 3, 4, 5)
Just like with lists, you can access values in a tuple by entering an index in square brackets.
